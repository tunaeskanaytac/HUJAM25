# Levels
> The second part is essentially a copy of hotline miami. A procedural generated map with enemies in certain places. The player will have to kill all the enemies to proceed through levels. The player will loot items with attributes that will help them to defeat stronger enemies. There will be no inventory system in the walkthrough. There will be safehouses in the map and player will have to be in that place to customize, upgrade and change their items.
